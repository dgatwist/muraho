## Install

```bash
pip install muraho
```

## Update

```bash
pip install --upgrade muraho
```

## Usage

```bash
usage: muraho [-h] [-i] [-s]

A Pentest Framework

optional arguments:
  -h, --help     show this help message and exit
  -i, --info     gets fsociety info
  -s, --suggest  suggest a tool
```


## Developing

```bash
git clone https://github.com/dgatwist/muraho.git
pip install -e ".[dev]"
```
